## Usage:
```
# Install dependencies
npm install

# Install client dependencies
cd admin-prac
npm install
cd ..

# Run client and server (Visit http://localhost:3000)
npm run dev

# Run server only (:5000)
npm run server

# Run client only (:3000)
npm run client
```
